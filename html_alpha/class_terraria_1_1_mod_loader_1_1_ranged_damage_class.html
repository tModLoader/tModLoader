override float Terraria.ModLoader.RangedDamageClass.GetBenefitFrom ( DamageClass  damageClass)
protectedvirtual

This lets you define the classes that this DamageClass will benefit from (other than itself) for the purposes of stat bonuses, such as damage and crit chance. Returns 0 in all cases by default, which does not let any other classes boost this DamageClass.

Parameters
damageClassThe DamageClass which you want this DamageClass to benefit from statistically.
