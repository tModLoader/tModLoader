◆ IsTallHat

bool [] ArmorIDs.Head.Sets.IsTallHat = Factory.CreateBoolSet(14, 56, 114, 158, 69, 180)
static

If true for a given Head, then that equip will allow a taller hat to render.
Defaults to false.

This is used by, for example, wizard hats and gladiator helmet. Without this setting, some animation frames would render cropped at the top.
