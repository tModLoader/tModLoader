Detailed Description

These sets determine which vanilla weapons belong to which prefix categories.

Modded items should not use these as modded item prefix categories are derived automatically from their Item.DamageType and/or ModItem.MeleePrefix/ModItem.MagicPrefix/ModItem.RangedPrefix/ModItem.WeaponPrefix usage instead.

ItemsThatCanHaveLegendary2, however, can be used by modded items if needed.
